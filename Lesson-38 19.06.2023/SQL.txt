SQL - Structured Query Language - язык структурированных запросов

Столбцы называют полями (атрибутами), они содержат информацию про свойства объекта

Строки называют записями(редко кортежами), они содержат значения конкретных свойств

SELECT [ALL | DISTINCT] (* | столбец_1 [, столбец_n])
FROM таблица_1 [, таблица_n]
WHERE условие
	[AND | OR]
 	COLLATE NOCASE
	выражение [NOT] BETWEEN(начало AND конец)
	выражение [NOT] LIKE шаблон_строки
		% - любое количество любых символов
		_ - один любой символ
	выражение [NOT] GLOB регулярное выражение
		* - любое количество любых символов
		? - соответствует любому одному символу
		. - соответствует любому одиночному символу
		[символы] - соответствует любому символу из списка
		[начальный_символ-конечный_символ] - соответствует любому символу из диапазона
		^ - этот символ используется в начале [] скобок и переворачивает смысл скобки. Например: выражение [^0-9] соответствует любому символу кроме цифр от 0 до 9.
	выражение IS [NOT] NULL - возвращает список всех вхождений, у которых выражение пустое / не пустое
	выражение IN (список_значений) - возвращает список всех вхождений, значения которых попадают в список
